# Atylus — Landing Page

Stack: React + Vite + Tailwind CSS (frontend) 

## Desenvolvimento local (sem Docker)

Frontend:
```bash
  npm install
  
  npm run dev        
```

---


## Personalizações rápidas

Trocar cores (src/index.css):

```css
  --color-ink: #0f0e0d
  
  --color-paper: #f5f2ee
  
  --color-accent: #c8a96e
  
  --color-accent-dark: #8b6914
```

Textos, cases e depoimentos:
  Tudo em arrays no topo de cada componente.

# build do projeto
```bash
npm run build
```

# Deploy 
```bash
npm run deploy
```

Isso cria/envia automaticamente a branch gh-pages.
